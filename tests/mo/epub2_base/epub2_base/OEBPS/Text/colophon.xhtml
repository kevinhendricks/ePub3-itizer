This Audio-eBook was crafted by ReadBeyond to celebrate the release of Menestrello, a free app for reading+listening Audio-eBooks in EPUB 3 reflowable format.

If you agree with us that synchronous highlighting and tap-to-play are useful features, we request you to urge the developers of your favourite reading system to implement the official EPUB 3 Media Overlay specification.

The text of this Audio-eBook is a revised version of the text available from Project Gutenberg.

The audio tracks were extracted from the reading by Chris Hughes for the LibriVox Project.
This Audio-eBook contains 5 audio tracks in MP3 format, 64 Kbps, 44 KHz, Stereo, for a duration of 4'.
If you want to extract the audio tracks, unzip this EPUB file and look inside the OEBPS/Audio/ directory.

The cover image is an original elaboration of a picture of William Shakespeare, whose digital scan was downloaded from Wikipedia.

Although we performed an automatic sanitization of the source materials, the text might still contain errors, mispelled words, and typos. Moreover, we had to apply a noise reduction filter to a dozen of audio tracks to remove a constant background hissing. Despite these imperfections, we hope that this Audio-eBook might prove useful and enjoyable.
